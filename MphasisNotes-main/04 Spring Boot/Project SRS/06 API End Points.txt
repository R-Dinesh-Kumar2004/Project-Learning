# 🔐 Auth Module

| Method | Endpoint              | Purpose           |
| ------ | --------------------- | ----------------- |
| POST   | /api/v1/auth/register | Register new user |
| POST   | /api/v1/auth/login    | Login user        |

# 👤 Users Module

| Method | Endpoint           | Purpose        |
| ------ | ------------------ | -------------- |
| GET    | /api/v1/users      | Get all users  |
| GET    | /api/v1/users/{id} | Get user by ID |
| PUT    | /api/v1/users/{id} | Update user    |
| DELETE | /api/v1/users/{id} | Delete user    |

### 🔎 Product Filtering

| Method | Endpoint                                     | Purpose               |
| ------ | -------------------------------------------- | --------------------- |
| GET    | /api/v1/products?category=Electronics        | Filter by category    |
| GET    | /api/v1/products?minPrice=1000               | Filter by min price   |
| GET    | /api/v1/products?minPrice=1000&maxPrice=5000 | Filter by price range |
| GET    | /api/v1/products?name=mobile                 | Search by name        |

### 🖼 Product Image

| Method | Endpoint                    | Purpose              |
